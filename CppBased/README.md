# EXE_Analyzer
*"Follow the message."*
基于Ring3 Inline Hook的程序行为分析工具
较之于python版本，运行效率更高，行为信息获取更彻底

# 功能
- [x] 反调试
- [x] 文件类API调用跟踪
- [x] 进程类API调用跟踪
- [x] 网络类API调用跟踪
- [x] 注册表类API调用跟踪
- [ ] 文件沙箱
- [ ] 恶意行为拦截
- [ ] 杂项API调用跟踪
- [ ] 基于PyQt5的GUI
